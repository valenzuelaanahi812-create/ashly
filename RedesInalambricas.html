Redes Inalámbricas

Conectividad global sin cables.

¿Qué es una red inalámbrica?

Es un sistema de comunicación de datos flexible que se implementa como extensión de, o como alternativa a, una red local cableada mediante tecnología de radiofrecuencia.

Estándar IEEE 802.11

Define cómo se transmiten los datos en el aire (Wi-Fi).

Puntos de Acceso (WAP)

Dispositivos que emiten la señal para que otros se conecten.
